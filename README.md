# Text-Extractor-App-Flutter
- A Text Extractor App From Image !

## ⚙️ Tech-Stack
The application is built entirely with Flutter. For the detection part, I have made use of Google ML Kit.

## Libraries

- Main Library Used in App 
    - Provider
    - Image Picker
    - Google ML Kit
    - Image Cropper
    - Google Fonts
