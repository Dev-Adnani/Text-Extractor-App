# Text-Extractor-App-Flutter
- A Text Extractor App From Image !
- Used Google ML Kit To Scrap Text From Image 

## Libraries

- Main Library Used in App 
    - Provider
    - Image Picker
    - Google ML Kit
    - Image Cropper
    - Google Fonts
